# MDN good-semantics example

This example recreates the MDN good-semantics HTML sample as ElementSpec and
SemanticNode data.

Sources:
- [MDN article](https://developer.mozilla.org/en-US/docs/Learn_web_development/Core/Accessibility/HTML)
- [Live demo](https://mdn.github.io/learning-area/accessibility/html/good-semantics.html)

Files:
- `good_semantics.ts`: ElementSpec and SemanticNode representations
