## Keyboard-Only Navigation

1. Unplug the mouse or avoid using it.
2. Use `Tab` and `Shift+Tab` to navigate through focusable elements.
3. Verify focus visibility for each control.
4. Use `Enter` or `Space` to activate buttons and links.
5. Use `Escape` to close modals or dialogs.
6. Record any keyboard traps or unreachable elements.
